# Commands — 커스텀 슬래시 명령어

입력할 수 있는 슬래시 명령어들입니다. 평소 작업할 때는 따로 명령어를 입력하지 않아도 됩니다. 그냥 할 일을 설명하면 AI가 프로젝트 규칙에 따라 작업을 안전하게 계획하고, 검증하며, 커밋합니다.

## 포함된 내용
| 명령어 | 역할 |
|---|---|
| `measure.md` | 설정을 변경할 때 도움이 되는지 확인하기 위해 변경 전/후 수치를 기록합니다. |
| `analyze-failures.md` | 무엇이 잘못되었는지 파악하기 위해 반복되는 도구 오류를 요약합니다. |
