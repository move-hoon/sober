# Commands — Custom slash commands

These are the slash commands you can type. For normal work, you don't need to run a command — just describe the task and the AI follows the project rules to safely plan, verify, and commit changes.

## What's here
| Command | What it does |
|---|---|
| `measure.md` | Record before/after numbers when you change your setup to see if it helps. |
| `analyze-failures.md` | Summarize repeated tool errors to figure out what went wrong. |
