# Codex

Codex-facing visibility lives here. Runtime hook/rule sources stay under `.sober/codex/` and are merged or linked into `~/.codex` by `sober install`.

- `AGENTS.md`: explains how Codex receives the Sober spine
- `config.toml.example`: optional local Codex configuration example

Do not put active Sober hook templates here; keep them in `.sober/codex/`.
