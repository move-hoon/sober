# Rules — 항상 켜져 있는 프로젝트 규칙

이 규칙들은 매 세션마다 자동으로 로드됩니다. AI가 안전하고 일관되게 작업할 수 있도록 안내합니다.

## 포함된 내용
| 파일 | 역할 |
|---|---|
| `critical-actions.md` | 위험한 명령어(`rm -rf`나 `git push --force` 등)를 실행하기 전에 사용자가 먼저 검토할 수 있도록 멈춥니다. |
| `security.md` | 안전한 코딩 관행(예: 비밀번호 하드코딩 금지)을 지키도록 합니다. |
| `code-style.md` | 프로젝트 전체의 코드 스타일을 일관되게 유지합니다. |

이 규칙들을 쓰기 위해 별도의 명령어를 실행할 필요는 없습니다. 항상 활성화되어 있습니다.
