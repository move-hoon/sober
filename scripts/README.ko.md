# Scripts — 로컬 커맨드라인 스크립트

내 PC에서 반복적인 작업을 처리해주는 작은 스크립트들입니다. AI가 직접 코드를 읽고 생각하는 대신, 평범한 커맨드라인 도구가 지루한 일을 처리하게 만들어 AI 예산을 절약합니다.

## 포함된 내용
| 파일 / 폴더 | 역할 |
|---|---|
| `verify.sh` | 프로젝트 타입을 파악하고 알맞은 빌드나 테스트 명령어를 실행합니다. |
| `analyze-failures.sh` | 무엇이 잘못되었는지 볼 수 있도록 도구 실패 로그를 요약합니다. |
| `hooks/` | Claude Code 훅과 대부분의 Codex 훅이 쓰는 작은 스크립트들입니다. |
| `codex-hooks/` | 입력 형태가 조금 다른 Codex 이벤트용 작은 어댑터입니다. |
| `runtime/` | `verify.sh`가 여러 언어를 빌드하는 방법을 알려주는 헬퍼 스크립트들입니다. |
